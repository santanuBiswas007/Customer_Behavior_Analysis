# 📊 Customer Behaviour Analysis

## 📌 Project Overview
This project analyzes customer purchasing behaviour using transaction data to uncover patterns, trends, and business insights.<br>
The objective is to help businesses make data-driven decisions by understanding customer habits and purchasing trends.<br>

## 🎯 Objectives
Identify customer purchase patterns<br>
Find high-value customers<br>
Analyze product and transaction trends<br>
Provide insights through dashboards<br>

## 🛠️ Tools & Technologies
🐍 Python – Data cleaning and preparation<br>
🗄️ SQL – Transaction and customer analysis<br>
📊 Power BI – Dashboard and visualization<br>
📑 PowerPoint – Presenting business insights<br>

## 🔄 Project Workflow

### 🐍 Data Preparation (Python)
Cleaned and structured raw data<br>
Handled missing values<br>
Created customer metrics for analysis<br>

### 🗄️ Transaction Analysis (SQL)
Revenue analysis by customer<br>
Product purchase trends<br>
Customer purchase frequency<br>
Customer segmentation<br>

### 📊 Dashboard Development (Power BI)
Sales trends visualization<br>
Top products analysis<br>
Customer segmentation dashboard<br>
Key performance indicators (KPIs)<br>

### 📑 Insight Presentation
Key findings summarized in PowerPoint<br>
Business recommendations for stakeholders<br>

## 💡 Key Insights
A small percentage of customers generate most revenue<br>
Certain products drive repeat purchases<br>
Customer buying behaviour changes over time<br>

## 🚀 Skills Demonstrated
Data Cleaning<br>
SQL Data Analysis<br>
Dashboard Development<br>
Business Insight Communication<br>
